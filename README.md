# Movie App 🎬

A responsive Movie App built using HTML, CSS, and JavaScript. The application fetches movie data from an external API and allows users to browse popular movies and search for their favorite titles.

## Features

* View popular movies
* Search movies by title
* Display movie ratings
* Responsive user interface
* Dynamic content rendering using JavaScript
* API integration with asynchronous data fetching

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
